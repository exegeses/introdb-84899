# Eliminar registros
 > Para eliminar registros dentro de una tabla utilizamos el comando **DELETE**
 > Debemos especificar mediante un filtro cuál es el registro que queremos eliminar 

> Sintaxis:

    DELETE FROM tabla
      WHERE columnaPrimaryKey = valorPK;

> Ejemplo:

    DELETE FROM proveedores  
       WHERE idProveedor = 15;
